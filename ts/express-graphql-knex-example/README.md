# Express & Graphql

A simple app using typescript, express, and graphql

## Setup

- populate `.env` file. template in `dev.env`
- `make install`

## Migrations (dev)

- `make migrate`

## Startup (dev)

- `make dev`

## Startup (prod)

- `make build`
- `make run`
