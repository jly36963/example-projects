# Vue / Nuxt

A starting point for Vue/Nuxt with TailwindCSS.

## Setup

- `npm i`

## Startup (dev)

- `npm run dev`

## Startup (prod)

- `npm run build`
- `npm start`

## Generate Static (if applicable)

- `npm run generate`

## Nuxt

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
