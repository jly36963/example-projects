# Vue / Nuxt / Express / D3

A simple Vue/Nuxt app with D3 usage and a custom server (Express).

## Setup

- `npm i`

## Startup (dev)

- `npm run dev`

## Startup (prod)

- `npm run build`
- `npm start`

## Nuxt

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
