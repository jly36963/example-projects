# Next D3

A SSR application that uses Next (SSR), React (SPA), Express (API), and D3 (data visualization).

## Setup

- `npm i`

## Startup (dev)

- `npm run dev`

## Startup (prod)

- `npm run build`
- `npm start`

## Reference

Much of the D3 code is based on examples from [Observable](https://observablehq.com/@d3/)
